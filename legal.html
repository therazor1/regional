  1. Manual de Uso del Servicio de la Aplicación

El objetivo de Sueño Seguro es realizar un monitoreo detallado de las horas de sueño de los Usuarios diariamente, antes del inicio de la ejecución del servicio de transportes contratado por PRIMAX. En ese sentido, la aplicación tiene como objetivo determinar si el Usuario se encuentra “APTO” para poder brindar el servicio de transporte en base a la información que se recopila a través de la aplicación “Zepp Life” y la pulsera “Smart Band”. En esa línea, para poder conseguir la información y data necesaria para determinar la clasificación del Usuario, este deberá seguir las siguientes instrucciones:

  1. Un (01) día antes del Inicio de un Servicio de Transporte; el Usuario, antes de dormir, deberá colocarse la pulsera “Smart Band” (la cual deberá vincularse con “Zepp Life” previa sincronización con la aplicación “Sueño Seguro”) para que recopile las horas que el Usuario le dedica a dormir antes de iniciar con el Servicio de Transporte y determinar si se encuentra APTO para ejecutar el servicio de transporte.
  1. Si para la ejecución total del Servicio de Transporte se tarda dos o más días, el Usuario deberá colocarse la pulsera “Smart Band” cada vez que se vaya a dormir para que la aplicación recopile la información y pueda determinar si se el Usuario se encuentra “APTO” para realizar el Servicio de Transporte.
  1. El Usuario, antes de iniciar o seguir con el servicio de Transporte, deberá verificar en la Aplicación si se encuentra “APTO” para el servicio de Transporte y en caso, este tenga una calificación desfavorable (“OBSERVADO” o “NO APTO”), deberá comunicar a su supervisor y esperar sus instrucciones.
